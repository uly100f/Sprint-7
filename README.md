# Vehicle Sales Dashboard

## Descripción

Este proyecto consiste en una aplicación web desarrollada con Streamlit para explorar un conjunto de datos de anuncios de venta de vehículos en Estados Unidos.

La aplicación permite visualizar la información mediante gráficos interactivos creados con Plotly.

## Funcionalidades

- Visualización de un histograma de la columna **odometer**.
- Visualización de un gráfico de dispersión entre **price** y **odometer**.
- Interfaz interactiva mediante botones o casillas de verificación.

## Tecnologías utilizadas

- Python
- Pandas
- Plotly
- Streamlit
